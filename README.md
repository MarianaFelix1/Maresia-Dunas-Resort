## 🚀 Funcionalidades Principais (CRUD & Login)

* **🔐 Tela de Login:** Autenticação com design moderno utilizando vídeo de fundo responsivo.
* **📊 Dashboard de Controle:** Visualização rápida dos quartos e status de ocupação através de uma barra lateral (Sidebar) integrada.
* **🛏️ Gerenciamento de Reservas (CRUD):** Criação, listagem e atualização de hóspedes e quartos.
* **🍹 Controle de Frigobar:** Lançamento automatizado de itens de consumo por hóspede com cálculo de valor total em tempo real.


## 🛠️ Tecnologias Utilizadas

O projeto foi estruturado dividindo as responsabilidades entre Front-End e Back-End:

| Camada | Tecnologia | Descrição |
| :--- | :--- | :--- |
| **Front-End** | HTML5 / CSS3 / JavaScript | Estruturação, estilização global, animações e manipulação do DOM. |
| **Back-End** | Node.js / Express | Criação da API REST e gerenciamento das rotas do sistema. |
| **Banco de Dados** | MySQL | Armazenamento relacional dos dados de usuários, quartos, hospedes, resevas, financeiro . |
| **IA** | Assistente de Engenharia | Utilizada como ferramenta de suporte para debugar rotas e otimizar a lógica do CRUD. |
